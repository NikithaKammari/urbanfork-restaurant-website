# UrbanFork – Restaurant Website

UrbanFork is a responsive restaurant website built using **HTML, CSS, Bootstrap 5, and JavaScript**.
The project demonstrates modern UI design for a restaurant, including menu display, image gallery, and reservation form.

## Live Demo

https://nikithakammari.github.io/urbanfork-restaurant-website/

## Project Overview

UrbanFork is a responsive restaurant website designed to showcase food items, restaurant services, and menu offerings in an attractive and user-friendly interface. The website allows visitors to explore different food categories, view images of dishes, and learn about the restaurant’s specialties.

The project focuses on building a modern front-end interface using HTML, CSS, Bootstrap 5, and JavaScript. It demonstrates responsive design principles so the website works smoothly on desktops, tablets, and mobile devices.

This project was created as part of frontend development practice to improve UI design skills, layout structuring, and responsive web development.


## Features

* Responsive restaurant landing page
* Food menu showcase
* Image carousel/gallery
* Reservation / booking form
* Toast notifications
* Modern UI with Bootstrap 5
* Mobile-friendly design

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
